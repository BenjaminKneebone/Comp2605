Skill Development Aims

We have made the decision that we wish to enhance are skills in Asynchronous web server communication and updates and Creating a responsive user interface with HTML5, CSS & Javascript. These were chosen as we are keen sports fans and these techniques are widely used across websites to react to real time sporting events.

Project Outline

Figure 3: Official Premier League Fantasy Football

Our intention is to develop a fantasy football web application prototype. Fantasy Football is an online competition within which users select a team of football players from real teams. The user then receives points based on their players' performances in real games (for example, 5 points if the player scores a goal). It is an extremely popular activity amongst groups of friends and colleagues and has many online variations ran by different newspapers and other companies. Our proposal stems from our mutual enjoyment of football, and our knowledge (through participation) of existing fantasy football applications. Due to the real time nature of retrieving live updates from sport, this is an appropriate context to apply asynchronous web server communication. Futhermore existing applications demonstrate a static interface, with updates only made to numerical values on the screen (such as those in figure 3). Instead, we propose to have a dynamic interface with more visuals in response to events. This project will also demand many of the other skill areas outlined in the coursework specification such as Database transcations, Input Data Validation and Web Patterns & Frameworks.
